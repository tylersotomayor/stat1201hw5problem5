This exercise from Homework 5 of STAT 1201 | Introduction to Calc-based Statistics (Summer 2025, Jonas Mikhaeil) explores how to use indicator variables in a linear regression model to predict student test scores based on the following regressors:

  • Pre-test scores,

  • Sex (binary: male or female),

  • Grade level (6, 7, or 8).

We compare two different ways to include grade in the model. First, as a numeric (linear) predictor. Second, as a categorical variable (with indicator variables).

Explicit answers to problem 5(a) and 5(b) can be found under Model A and Model B

Simulate Data

Before we begin, we will first simulate a dataset of student records to test our models and validate our code.

# Load necessary package
library(tidyverse)  # Provides tools like dplyr, ggplot2, and tibble

# Set seed so results are reproducible each time you knit the document
set.seed(123)

# Create a fake dataset with 100 students
mydata <- data.frame(
  pre_test = runif(100, 0, 100),             # Random scores between 0100
  sex = sample(0:1, 100, replace = TRUE),    # 0 = female, 1 = male (binary coding)
  grade = sample(6:8, 100, replace = TRUE)   # Randomly assign grade level: 6, 7, or 8
)

# Now generate the outcome variable (test_score)
# We assume:
#   - Higher pre_test scores improve final scores
#   - Males score slightly higher on average (sex effect)
#   - Higher grades have slightly higher performance
#   - Add noise using rnorm() to simulate real-world randomness

mydata$test_score <- 0.5 * mydata$pre_test +     # strong effect of prior performance
                     5 * mydata$sex +            # moderate effect of sex (male = 1)
                     2 * mydata$grade +          # increasing grade level = higher score
                     rnorm(100, mean = 0, sd = 10) # random noise (residual variation)

Model A: Treating Grade as a Linear Variable

In this model, we treat grade as a numeric linear predictor assuming a linear relationship between grade and test score.

model_linear <- lm(test_score ~ pre_test + sex + grade, data = mydata)

The estimated regression model suggests that, holding pre-test scores and sex constant, a one-unit increase in grade level (e.g., from grade 6 to 7, or from 7 to 8) is associated with an average increase of approximately 3.80 points in the test score (p = 0.008). This effect is statistically significant at the 1% level, indicating strong evidence of a positive linear relationship between grade level and test performance in the simulated data.

Additionally, pre-test scores have a large and highly significant positive effect: for every one-point increase in pre-test score, the expected test score increases by approximately 0.51 points (p < 0.001). The binary variable for sex (coded as 1 for male, 0 for female) indicates that male students, on average, score about 5.58 points higher than female students, controlling for grade and pre-test performance (p = 0.014).

The overall model explains approximately 63.65% of the variance in test scores (R = 0.6365), with an adjusted R of 0.6251, indicating strong model fit.

Model B: Treating Grade as a Categorical Variable (Indicators)

To answer Problem 5(b) we use Rs built-in factor() function to treat grade as a categorical variable. R assigns dummy (indicator) variables for grade 7 and 8, allowing us to estimate separate effects for each grade level relative to grade 6 (the baseline category). We treat Grade 6 as the baseline (intercept includes grade 6). This model estimates the effect of being in grade 7 or 8 relative to grade 6. Note: this approach makes no assumption about the relationship between grade level and test scores being linear

model_indicators <- lm(test_score ~ pre_test + sex + factor(grade), data = mydata)

In this specification, grade level is modeled as a categorical variable with grade 6 serving as the reference category. The regression results indicate that, controlling for pre-test scores and sex, students in grade 8 score on average 7.70 points higher than students in grade 6 (p = 0.0088). This difference is statistically significant at the 1% level. Students in grade 7, by contrast, score on average 3.46 points higher than students in grade 6, although this estimate is not statistically significant (p = 0.1804), suggesting that the score difference between grades 6 and 7 is not reliably different from zero in this sample.

The effect of pre-test scores remains strong and significant: a one-point increase in pre-test score is associated with an average increase of 0.51 points in the test score (p < 0.001), nearly identical to the linear-grade model. The sex coefficient indicates that male students score on average 5.56 points higher than female students, holding other variables constant (p = 0.015).

This model explains approximately 63.66% of the variation in test scores (R = 0.6366), with an adjusted R of 0.6213. These values are nearly identical to those from the linear-grade model, indicating that the added flexibility of using indicators does not meaningfully improve model fit in this case likely because the relationship between grade and test score is already close to linear in the underlying data generating process!

Plots: Model Diagnostics and Validation of Model Assumptions

Finally I generate a couple plots to validate our code and check assumptions of our linear model specification (linearity, normality of residuals, homoskedasticity, no wild outliers)

# 2x2 plotting grid
par(mfrow = c(2, 2))

# Plot diagnostics for the linear grade model
plot(model_linear)

These built in diagnostic plots help us assess whether the assumptions of linear regression have been reasonably met. In the Residuals vs.Fitted plot, the points appear randomly scattered around the horizontal line, suggesting that the model captures a linear relationship without clear patterns or curvature. The Normal Q-Q plot shows that the residuals closely follow the diagonal line, indicating that they are approximately normally distributed. The Scale-Location plot displays a relatively even spread of residuals, supporting the assumption of constant variance across fitted values. Finally, the Cooks Distance plot reveals no extreme spikes, implying that no individual observation has an outsized influence on the model. Taken together, these diagnostics suggest that the model is well-specified and its estimates are reliable.
